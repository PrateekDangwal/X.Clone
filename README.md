# X Clone (Twitter Clone)

A responsive Twitter (X) Clone built using **HTML** and **Tailwind CSS**. This project recreates the modern X (Twitter) user interface, including a sidebar, feed section, trending panel, posts, videos, and responsive layouts.

## Features

* Responsive Design
* Left Navigation Sidebar
* Sticky Feed Navigation (For You / Following)
* Search Bar
* Trending Section
* Follow Suggestions
* Tweet/Post Cards
* Video Posts
* Hover Effects
* Mobile-Friendly Layout
* Tailwind CSS Styling

## Tech Stack

* HTML5
* Tailwind CSS
* Git & GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/PrateekDangwal/X.Clone.git
```

2. Navigate to project folder

```bash
cd X.Clone
```

3. Install dependencies

```bash
npm install
```

4. Start Tailwind build process

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```

5. Open `index.html` in your browser.
